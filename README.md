# YouTube Auto Loop (Opera GX) 🔁

A lightweight, fully native-looking browser extension for Opera GX that adds a dedicated **Loop Button** directly into the YouTube video player controls. 

Perfect for leaving your favorite songs, gaming soundtracks, or background videos on an infinite loop without right-clicking the video every time.

## ✨ Features
- **Native Integration**: Seamlessly blends into the YouTube player control bar, maintaining the exact dimensions and styling of native buttons.
- **Opera GX Aesthetics**: Lights up in the iconic Opera GX red (`#fa1e4e`) with a smooth micro-animation when activated.
- **Persistent State**: Automatically keeps your videos looping, even if YouTube's SPA (Single Page Application) architecture tries to reset the state.
- **Zero Configuration**: Just install and enjoy!

## 🚀 Installation (Developer Mode)
1. Open **Opera GX** and go to `opera://extensions`.
2. Toggle on **Developer mode** in the top right corner.
3. Click on **Load unpacked** and select the folder containing this repository.
4. Open any YouTube video, look at the bottom right controls, and click the new loop icon!
